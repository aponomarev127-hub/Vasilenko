# Random Password Generator

Автор: Данила Василенко

## Описание

Random Password Generator — это настольное приложение с графическим интерфейсом, разработанное на языке Python с использованием библиотеки Tkinter. Программа предназначена для генерации случайных и безопасных паролей с возможностью настройки их параметров.

Пользователь может выбрать длину пароля и определить, какие символы будут использоваться: буквы, цифры и специальные символы. Сгенерированные пароли сохраняются в истории, которая автоматически записывается в JSON-файл и загружается при запуске программы.

Данное приложение может использоваться для создания надёжных паролей при регистрации в сервисах и повышении уровня безопасности аккаунтов.

## Возможности

- генерация случайных паролей
- настройка длины пароля (ползунок)
- выбор состава пароля:
  - буквы
  - цифры
  - специальные символы
- отображение результата генерации
- сохранение истории паролей
- загрузка истории при запуске
- хранение данных в формате JSON
